# CampusRunner

CampusRunner is a campus errand marketplace web app with Student, Runner, and Admin dashboards.

## Quick start

```bash
npm install
npm run dev
```

## Demo auth
This build uses localStorage demo auth so routing and UI can be reviewed quickly without a backend.

## Demo accounts
- Admin: `admin@campusrunner.app` / `admin123`

Students and runners can sign up from the UI.
